# 🛰️ AI Radar Surveillance System

An AI-powered 2D Radar Surveillance System built using **Python** and **Pygame**. The project simulates a modern radar interface with aircraft tracking, radar sweep animation, HUD elements, and a modular architecture designed for future AI-based surveillance features.

## ✨ Features

- 📡 Real-time rotating radar sweep
- ✈️ Multiple aircraft simulation
- 🎯 Target detection and tracking
- 🟢 Radar glow effects
- 📊 Heads-Up Display (HUD)
- 🎨 Modern radar visualization
- 🧩 Modular Object-Oriented Design
- ⚡ Smooth 120 FPS rendering
- 🔧 Easily expandable architecture

## ▶️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/shubhamkardel-ai/AI-Radar-Surveillance-System.git
```

### 2. Open the project

```bash
cd AI-Radar-Surveillance-System
```

### 3. Install Pygame

```bash
pip install pygame
```

### 4. Run the application

```bash
python Main.py
```
